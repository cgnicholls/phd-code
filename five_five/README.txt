The file 'five_five_interpolation.m' verifies the computations for the (5, 5)-isogeny in the thesis. See the references to 'five_five_interpolation.m'.

You can run this by typing 'magma' at the command prompt, and then, once magma opens, type the command 'load "five_five_interpolation.m"'. Note that you need to be in the 'five_five' directory for this to work.
